##  Draw tides

Library for draw tides graph by data point from https://github.com/pwsdotru/worldtides-client


### Examples of data

````
Array
(
    [0] => Array
        (
            [dt] => 1782752400
            [date] => 2026-06-29T17:00+0000
            [height] => 0.553
        )

    [1] => Array
        (
            [dt] => 1782754200
            [date] => 2026-06-29T17:30+0000
            [height] => 0.422
        )

    [2] => Array
        (
            [dt] => 1782756000
            [date] => 2026-06-29T18:00+0000
            [height] => 0.26
        )

)
````

Where **height** is water height for date and time that are are **dt** (unix timestamp) and/or **date** (string date and time)
